## 🔍 常见问题排查

### 问题 1: 部署后仍然提示"Supabase 未配置"

**可能原因**：
- 环境变量名称拼写错误
- 环境变量值有多余空格或引号
- Vercel 缓存问题

**解决方法**：
1. 在 Vercel Dashboard 中重新检查环境变量
2. 确保名称完全一致：
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
3. 在 Vercel 项目 Settings → Git 中清理缓存并重新部署

### 问题 2: 注册成功但无法登录

**可能原因**：
- Supabase 的 confirm email 未关闭
- 用户状态未激活

**解决方法**：
1. 检查 Supabase Dashboard → Authentication → Providers
2. 确认 Email provider 的 "Confirm email" 选项已关闭
3. 在 Authentication → Users 中查看用户状态

### 问题 3: 数据不同步

**可能原因**：
- 用户未正确登录
- RLS 策略配置问题
- 网络问题

**解决方法**：
1. 检查浏览器控制台（F12）查看错误
2. 确认已登录（右上角显示邮箱）
3. 在 Supabase Dashboard → Database 检查 strategies 表
